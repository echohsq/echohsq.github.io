工欲善其事必先利其器,我们做前端的不用总是使用一些看起来很极客
的编辑器了,让我们来看一看强大的webstorm.

本文主要介绍一些要记的快捷键,详细内容可参照明河的入门指南

工程和界面

webstorm中的工程

  1. 收藏夹功能
    我们可以在层级很深但要经常访问的地方,右键设置到favorites中,
    可以进行快速的调用.
  2. 工程界面
    我们可以使用”alt + 1”来快速拆合工程界面.
  3. 构造器界面
    在此界面可以快速查找js类的函数和对象.

    默认配置下的常用快捷键

    查找/替代

    | 快捷键 | 说明 |
    | —– | —-: |
    | ctrl+shift+N | 通过文件名快速查找工程内的文件（必记）|
    | ctrl+shift+alt+N | 通过一个字符快速查找位置（必记）|
    | ctrl+F | 在文件内快速查找代码 |
    | F3 | 查找下一个 |
    | shift+F3 | 查找上一个 |
    | ctrl+R | 文件内代码替换 |
    | ctrl+shift+R | 指定目录内代码批量替换 |
    | ctrl+shift+F | 指定目录内代码批量查找 |
    | ctrl+R | 文件内代码替换 |
